ADEQUACY : 

Source text adherence is judged to the source text norms and meaning, in terms of how well the target text represent the informational
content of the source text.
 
Does the output convey the same meaning as the input sentence  ? Is the part of the message lost, added , or distorted.



FLUENCY :

The degree od adherence to the target text and target language norms, referring. For Example,
grammatical correctness and clarity. When judging fluency, the source text is not relevant.

Is the output good fluent English? 
This envolves both grammatical correctness and idiomatic word choices.
